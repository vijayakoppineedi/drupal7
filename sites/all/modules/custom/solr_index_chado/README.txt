Module to index the Tripal chado fields into Apache Solr

The module comes with a schema.xml and solrconfig.xml file which
must be used in your Solr installation.

This module was tested with Solr 4.10.x.

Copy every file from solr-conf to APACHE_SOLR/conf/

1 - Start Apache Solr
2 - Install module

This is module is depend on the search and apachesolr module.
